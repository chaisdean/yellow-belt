Colors and Formatting

This template includes six color themes. Colors are easily, globally
changed for the entire empleate by editing the plain text colortheme.css
with your choice. Instantly change the color of your website. Template
font sizes and colors, page buttons and menu colors and fonts are all set
in globall CSS files for easy formatting updates. You can also setup your
own custom color theme. If you are creating your ouwn theme some
graphics editing will be required.

Features

Plain text global menus, global sidebar areas, header and footers are
setup for easy website updates. The drop menus can be setup 3 deep for
larger websites. A sample multi-level menu file is included with the
download. Or any menu item can be set as a plain button with no drop
menu. The sidebar search can be switched off, setup as a Google search
or, 34 options are inluded for a safe search. The sidebar scroller is
displaying 3 basic HTML pages and is updated like any other template
page. The right sidebar can be switched off if not required. View the step by step
instructions.
